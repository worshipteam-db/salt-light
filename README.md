# Salt & Light

Salt & Light is a gamified Christian personal growth and devotion web app built with React, Vite, Supabase, and TailwindCSS.

Users can:
- Create a character
- Complete goals and devotions
- Gain XP and faith
- Unlock skills and equipment
- Track spiritual growth
- View recent activity and progression
- Use the app as a mobile-friendly PWA

---

# ✨ Features

## Character System
- RPG-inspired progression
- XP and leveling
- Faith points
- Job/class system
- Skill tree
- Equipment system
- Character customization

## Goals System
- Create daily and long-term goals
- Gain XP for completing goals
- Goal archive/history
- Progress tracking

## Devotion System
- Daily devotion entries
- Bible verse selection
- Streak tracking
- Faith and XP rewards
- Monthly devotion progress grid

## Activity Log
- Recent actions feed
- XP and faith history
- Equipment purchases
- Level-up tracking

## Authentication
- Supabase Auth
- Email/password login
- Persistent sessions

## Responsive Design
- Mobile-first UI
- Installable PWA experience
- Optimized for phones and tablets
